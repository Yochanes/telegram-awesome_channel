Если кому вдруг интересно, то ваш роутер скорее всего можно настроить так, чтобы при обращении к определенным доменам, маршрутиризатор отправлял на определенный ip внутри вашей домашней сети.
Например, мне для тестирования моб. приложения надо было чтобы по вайфай телефон соединялся с домашним компом.
Я зателнетился, и прописал маршрут так:
telnet 192.168.1.1
где 192.168.1.1 - ip вашего домашнего роутера-шлюза.
вводим логин-пароль от роутера, и прописываем:
ip host foobar.com 192.168.1.39

теперь если тел. подключен по вайфай к роутеру, то при заходе на foobar.com будет открываться страница с хоста (дом. комп) 192.168.1.39
Такая схема работает в Zyxel kenetic lite и скорее всего в большинстве нормальных роутерах. См. документацию, читайте форумы ))

в теории можно просканить (nmap) сетку на поиск таких же домашних роутеров с логином-паролем admin admin, зателнетиться и прописать ip host vk.com <ip вашего фишингового сервера> и ловить аккаунты vk соседа(ки) =)
